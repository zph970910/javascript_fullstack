# npm i webpack webpack-cli --save-dev

# dist 存放打包之后的文件

# src 存放入口文件

# webpack.config.js // webpack的配置文件


webpack中的（module）用来决定如何处理项目中的不同类型模块

webpack 模块支持语法：
  es6 import 语句
  Commonjs require() 语句
  AMD define 和 require 语句
  css/sass/less 文件中的@import 语句
  样式（url(...)）或者html文件(<img src="...">)中的图片连接
